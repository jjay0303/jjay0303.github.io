## ora-00942: table or view does not exist 에러 해결법

#### 1. 진짜 테이블이나 뷰가 없음 => 테이블,뷰 이름 오타 확인

#### 2. 해당 테이블에 권한이 없음 => 테이블에 권한 부여

#### 3. DB에 연결이 안됨 => root-context.xml에서 DB 계정 확인

#### 4. 테이블 앞에 소유자 명시하여 select => select * from 소유자.테이블명;
